# Web Scraper using flask

## Installation

Install with pip:

```
$ pip install -r requirements.txt
```


## Flask Application Structure 
```
.
├── README.md
├── main.py
├── requirements.txt
├── static
│   ├── css
│   │   ├── index.css
│   │   ├── loading.gif
│   │   ├── result.css
│   │   └── style.css
│   └── js
│       └── script.js
└── templates
    ├── index.html
    └── result.html
```

## Run the application

```
$ python main.py
```
