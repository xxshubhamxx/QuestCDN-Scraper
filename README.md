# Web Scraper using flask

## Installation

Install with pip:

```
$ pip install -r requirements.txt
```

## Flask Application Structure

```
.
├── README.md
├── app.py
├── requirements.txt
├── geckodriver.exe
├── Dockerfile
├── static
│   ├── css
│   │   ├── index.css
│   │   ├── loading.gif
│   │   ├── result.css
│   │   └── style.css
│   └── js
│       └── script.js
└── templates
    ├── index.html
    └── result.html
```

## Run the application

```
$ python app.py
```
